# Hospitality Management System Using SQL

## Project Overview
This project is a SQL-based Hospitality Management Analytics System designed to manage and analyze hotel operations. It demonstrates how SQL can be used to organize, query, and analyze hospitality data, including customer information, hotel details, bookings, payments, and revenue. The project focuses on extracting meaningful business insights using SQL queries and database management concepts.

## Tools & Technologies
- MySQL
- SQL
- Relational Database Management System (RDBMS)

## Dataset
The project uses a Hospitality Management dataset containing:
- Customers
- Hotels
- Bookings
- Payments
